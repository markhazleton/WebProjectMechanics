More style, more fun!

Welcome to Variant Fashion, a free website template by Andreas Viklund. This is a regular XHTML/CSS website template which can be used to build your own website - or to build a theme for your favorite CMS or blog engine. This template is released as free code, which means that you are free to use it for any purpose and in any way you may want to without any obligations. I kindly ask that you leave the credit text and link (the one that says "Design by Andreas Viklund" in the footer) since that would be a nice way of giving something back to me. But the decision is yours, as always with templates from andreasviklund.com.

Read more »

Template features:

  • Multiple layout options.
  • Multiple design options.
  • Valid XHTML 1.0 and CSS2.
  • No tables used in the layout.
  • Perfectly readable even without CSS.
  • No-scripts image rollover menu (a:hover).
  • Low load size.

Blog/CMS theme versions

Are you looking for a WordPress theme version of this template? Or maybe a version that is ported for some other CMS or blog engine? Then make sure to check the Variant Fashion page on andreasviklund.com to see if there are any version available that you can use. If you make a port yourself, please let me know so I can link to it from the page.
